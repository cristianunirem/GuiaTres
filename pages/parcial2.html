Preguntas del Segundo Parcial


13. ¿Para qué se utilizan los siguientes comandos?

Comando Descripción y Uso
npm install Descarga e instala todas las dependencias y bibliotecas especificadas en el archivo package.json.
npm run web Inicia el servidor de desarrollo local para previsualizar la aplicación en el navegador.

14. Estás desarrollando una app y necesitas actualizar bibliotecas y dependencias para resolver conflictos entre versiones. ¿Qué comando debes utilizar?


  • a) run update
  • b) npm install
  • c) install libs-dev
  • d) install -all

Justificación: npm install sincroniza las versiones del proyecto para resolver conflictos de dependencias entre bibliotecas.


15. Selecciona el comando correcto para ejecutar la aplicación en un servidor de pruebas.


  • a) npm run web
  • b) npx run server
  • c) run dev-server
  • d) run npm web

Justificación: Es el comando estándar en el ecosistema de Expo para levantar la versión web del proyecto.


16. Selecciona la opción que representa un error común al instalar el entorno de ejecución.


  • a) Ejecutar archivos sin permisos de administrador.
  • b) Verificar la instalación con la terminal de comandos.
  • c) Descargar archivos desde páginas oficiales.
  • d) Ejecutar archivos como administrador.

Justificación: La instalación de entornos de ejecución requiere modificar directorios del sistema que están protegidos.


17. Son verdaderas (1) o falsas (0) las siguientes afirmaciones:


  • 1) Las vistas conforman los caminos de navegación: 0 (Falso)
  • 2) flexDirection define la orientación de elementos: 1 (Verdadero)
  • 3) Los props permiten crear componentes reutilizables: 1 (Verdadero)
  • 4) La documentación sólo proporciona instalación: 0 (Falso)
  • 5) Al agregar un archivo, este se convierte en ruta: 0 (Falso)

18. Un desarrollador quiere asegurar que su app funcione sin errores de navegación después de configurar el router. ¿Qué acción debe hacer para evitar conflictos de compilación?


  • a) Borrar el caché del proyecto.
  • b) Borrar el caché del servidor de pruebas.
  • c) Renombrar todos los archivos.
  • d) Renombrar el archivo ejecutable.

Justificación: Limpiar el caché obliga a la aplicación a recargar la configuración del router desde cero, eliminando errores de rutas antiguas.


19. ¿Qué tipo de vista se utiliza para organizar otras vistas dentro de la interfaz gráfica de la aplicación?


  • a) Imagen
  • b) Organizador
  • c) Contenedor
  • d) Pila

Justificación: En UI, un "Contenedor" (como el componente View) es el encargado de agrupar y organizar otros elementos hijos.


20. ¿Cuál es la palabra reservada del framework para permitir que un componente reciba información de otros componentes?


  • a) params
  • b) getter
  • c) props
  • d) setter

Justificación: Las props son el canal oficial para pasar datos entre componentes de forma descendente.


21. ¿Cuál es la utilidad de las variables translateX y translateY dentro del objeto “Arrastrar”?


Respuesta: Controlan el movimiento en los ejes X e Y.

Justificación: Permiten calcular la traslación de un objeto en el plano bidimensional durante una interacción táctil.


22. En la app "Pegatinas". ¿Cuál es la función principal del componente "Selector de imágenes"?


  • a) Acceder a la galería del dispositivo para elegir una imagen.
  • b) Establecer la imagen predeterminada en el visor de imágenes.
  • c) Reemplazar el icono del botón "Escoger una imagen".
  • d) Dejar a libre elección del usuario la imagen a elegir.

Justificación: Este componente vincula la app con la biblioteca de medios nativa del teléfono móvil.


23. En la app "Pegatinas", ¿Qué opción debe elegir el usuario si quiere trabajar con una imagen almacenada en su dispositivo?


  • a) Escanear con la cámara.
  • b) Usar esta imagen.
  • c) Elegir una imagen.
  • d) Importar una imagen.

Justificación: Es la etiqueta del botón que acciona el flujo de selección de archivos almacenados localmente.


24. ¿Cuál es la diferencia entre los comandos "eas login" y "eas init"?


Comando Función
eas login Autentica al desarrollador con su cuenta de Expo Cloud.
eas init Configura el proyecto y le asigna un ID único en los servicios de EAS.

25. ¿Por qué la compilación es un requisito para compartir la aplicación con diferentes dispositivos móviles?


  • a) Crea una cuenta de desarrollador.
  • b) Ajusta las bibliotecas para cada SO.
  • c) Porque garantiza la compatibilidad e instalación sin errores.
  • d) Porque verifica errores de programación.

Justificación: La compilación empaqueta el código y recursos en un formato que el hardware móvil puede ejecutar de forma nativa.
